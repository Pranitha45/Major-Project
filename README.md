# Major-Project
This Repo Consists of our major Project Called Anomaly Detection in Network Traffic.Our Project is a Research based project
Abstract:
With the current age of the digital era, network security is given priority because of the rising cyber-attacks
and malicious incidents. This project is for creating a desktop application for Network Traffic Anomaly
Detection using machine learning and deep learning. The application will classify and determine network
traffic as normal and other types of attacks like Neptune, Teardrop, and Port Sweep. To ensure effective
detection, the application utilizes four strong algorithms: 1. Support Vector Machine (SVM), 2. Random
Forest (RF), 3. Deep Neural Network (DNN), and 4. Extreme Learning Machine (ELM). All the models
are trained from labelled network traffic data to ensure effective anomaly detection. Performance of each
algorithm is measured based on general classification metrics like Accuracy, Precision, and Recall. The
primary objective of the project is to implement a powerful yet friendly tool with the ability to identify
suspicious activity, thus boosting overall network system security. The desktop application showcases how
modern machine learning models can be applied to promote cyber security protection

Tools & Technologies: Python, Jupyter Notebook, Scikit-learn, TensorFlow/Keras, Pandas, NumPy, Matplotlib, Seaborn

Project Definition: Developed a desktop-based Network Traffic Anomaly Detection system using Machine Learning and Deep Learning to classify network traffic as normal or malicious (Neptune, Teardrop, Port Sweep), achieving high detection accuracy and improving network security.

Role & Responsibilities: Performed data preprocessing and feature engineering on network traffic datasets, trained and evaluated multiple models (Support Vector Machine, Random Forest, Deep Neural Network, and Extreme Learning Machine), compared performance using Accuracy, Precision, and Recall metrics, and developed a user-friendly desktop application for real-time anomaly detection and cyber threat identification.
